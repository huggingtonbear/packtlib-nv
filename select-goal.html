What is a Skill Card?

Packt has been following what's important to developers for more than a decade. With more than 3,500 products used by millions of software professionals, we understand developer challenges, and empower them with the resources they need to build better software.

Today, the proliferation of tools and technologies can be difficult to navigate. That’s why we’ve developed Skill Cards - each one packages the best content you need to learn the most relevant skills.
